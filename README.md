# موقع مودار الساكني

منصة مودار هي تجربة واجهة عربية لإدارة المحافظ العقارية ومتابعة العملاء بمساعدة الذكاء الاصطناعي.

## تشغيل الموقع محليًا

> يعتمد تحميل الترويسات والتذييلات على جلب ملفات HTML جزئية، لذا يجب تشغيل الموقع من خادم محلي وليس عبر فتح الملفات مباشرة.

1. تأكد من وجود Node.js على جهازك.
2. من داخل مجلد المشروع نفّذ:

   ```bash
   npx live-server
   ```

3. سيفتح المتصفح تلقائيًا على العنوان `http://127.0.0.1:8080` (أو العنوان الظاهر في الطرفية). افتح أي صفحة مثل `index.html` للتحقق من الترويسة والتذييل المشتركين.

## بنية الملفات

- `assets/css/style.css`: الأنماط العامة والألوان باتجاه RTL.
- `assets/partials/header.html` و `assets/partials/footer.html`: الترويسة والتذييل المشتركان لجميع الصفحات.
- `assets/js/include.js`: سكربت تحميل الأجزاء المشتركة.
- `assets/js/app.js`: سلوكيات القائمة والتنقل السلس.
- الصفحات الرئيسية (11 صفحة) تتبع نفس الهيكل مع تضمين الترويسة والتذييل تلقائيًا.
